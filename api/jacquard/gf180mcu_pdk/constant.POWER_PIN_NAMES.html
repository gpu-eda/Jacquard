Power / ground pin name set used across the GF180MCU standard-cell and pad libraries, plus the wafer.space power-pad naming (DVDD/DVSS/AVDD/AVSS). Single source of truth for both pin-direction resolution (GF180MCULeafPins::direction_of) and the clock-tracing pin-enumeration skip in aig.rs. See PR #70 for context on the latter: post-#64 these names resolve to Direction::I, so the clock-tracer’s “any non-clock input pin ⇒ multi-input cell ⇒ clock gating” rule started tripping on every clkbuf cell that has wired power pins.
